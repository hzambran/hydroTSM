# San Martino, ts of daily precipitation.

Daily time series of precipitation, at station San Martino di Castrozza,
Trento Province, Italy, with data from 01/Jan/1921 to 31/Dec/1990.

## Usage

``` r
data(SanMartinoPPts)
```

## Format

zoo object.
