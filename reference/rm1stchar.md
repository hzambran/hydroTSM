# Remove First Character(s)

Deletes the first `n` character(s) of a character object.

## Usage

``` r
rm1stchar(x, n = 1)
```

## Arguments

- x:

  Character, e.g, each element may represent the name of a single
  gauging station.

- n:

  numeric, indicating the number of characters that have to be removed
  from the beginning of `x`

## Value

character object of the same length as `x`.
